# Oktara WareHouse

## 1- Server configuration
#### Open a terminal in a root of proyect
`cd server`

#### Get server Dependencies
`npm i`

#### Run server in development
`node index.js`

## 2- Client configuration
#### Open a terminal in a root of proyect
#### Go to Logistics-Company
`cd server`

#### Get server Dependencies
`npm i`

#### Run server in development
`npm start`
